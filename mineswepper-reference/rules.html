Mineswepper Ruleset

Objective: Uncover all cells that do not contain mines. If you reveal a cell with a mine, you lose the game.

Board Setup:

  • The board consists of a grid of cells, with a certain number of hidden mines randomly placed.
  • The number of mines is predetermined based on the difficulty level.

Revealing Cells:

  • Click on a cell to reveal what is underneath.
  • If the cell contains a mine, the game is over (you lose).
  • If the cell does not contain a mine, a number appears indicating how many adjacent cells contain mines.
  • If the revealed cell has no adjacent mines (number 0), all adjacent cells are automatically revealed.

Marking Mines:

  • You can mark a cell as a suspected mine by placing a flag on it.
  • Flags help keep track of potential mines, but you do not need to mark all mines to win.

Winning: The game is won when all cells that do not contain mines are revealed. Marking all mines correctly with flags is not a requirement for winning, but it can help keep track of dangerous spots.
